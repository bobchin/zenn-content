### C/C++ 開発環境構築

1.  VS Code インストール
1.  Mac と Linux のみ必要な環境のインストール
    - Mac  : ``` xcode-select --install ```
    - Linux: ``` sudo apt install python3 git tar build-essential ```
1. VS Code 拡張機能のインストール
    - "raspberry pi pico" で検索して、「Raspberry Pi Pico」拡張をインストール
        ※これで必要なものが自動的にインストールされる感じ

### CMakeLists.txt

CMake 用の設定ファイル

```c
#Set the mimum required version of CMake
#最低限必要なCMakeのバージョンを指定
cmake_minimum_required(VERSION 3.12)

#Pull in SDK (must be before project)
#必ず読み込む必要がある。パスなどの設定をしている。
include(pico_sdk_import.cmake)

#Set the project name, the programming language
#プロジェクト名とプログラミング言語を設定
project(project1 C CXX ASM)
set(CMAKE_C_STANDARD 11)     // C のバージョン
set(CMAKE_CXX_STANDARD 17)   // C++ のバージョン

#Initialize the SDK
#SDKの初期化。Picoの環境を読み込み。
pico_sdk_init()

#Add an executable target to be build from the source files
#出力する実行ファイル名と元となるソースを指定
add_executable(test test.c)

#Pull in our pico_stdlib which aggregates commonly used features
#ビルドに必要なライブラリの読み込み。最終実行ファイル名との関連付け。
target_link_libraries(test pico_stdlib)

#Enable USB output, disable UART output
#USB出力の有効化。UART出力の無効化。
#標準出力をUARTではなく、USBにしている。
pico_enable_stdio_usb(test 1)
pico_enable_stdio_uart(test 0)

#Create map/bin/hex/uf2 file etc.
#uf2ファイルの作成
pico_add_extra_outputs(test)
```
