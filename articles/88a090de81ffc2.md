### 詳細

```bash
# ツール(net rpc)のインストール
sudo apt-get install samba-common

# 実行
HOST=192.168.0.1
USER=administrator
PASS=password
net rpc shutdown -f -t 0 --ipaddress ${HOST} --user ${USER}%${PASS}
net rpc shutdown -f -t 0 -I ${HOST} -U ${USER}%${PASS}

net rpc shutdown -f -t 0 -r --ipaddress ${HOST} --user ${USER}%${PASS}
net rpc reboot -f -t 0 -I ${HOST} -U ${USER}%${PASS}
```

### Windows の設定

- レジストリ(UACのリモート制限を無効)

  ```ini
  HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System\LocalAccountTokenFilterPolicy = 1(DWORD)
  ```

- マイコンピュータ > ツール > フォルダオプション > ビュー
  - [x] 「**簡易ファイルの共有**」 をチェック外す

- コントロールパネル > セキュリティセンター > Windows ファイアウォール > 例外
  - [x] 「**ファイルとプリンターの共有**」にチェック

- Advanced tab > ICMP Settings
  - [x] 「受信エコー要求を許可する」にチェック
