## まとめ

どんな javascript ランタイム上でも動作するウェブアプリケーションフレームワーク

### インストール

```bash:bash
# インストール
npm create hono@latest my-app
cd my-app
npm install

# npm create hono@latest
# = npm init hono@latest (create は init の alias)
# = npm exec create-hono@latest


# ローカルサーバ実行
npm run dev
```

### まずはじめに

- シンプル

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { Hono } from 'hono'

  const app = new Hono()
  app.get('/', (c) => c.text('Hello Node.js!'))

  serve(app)
  ```

- 起動ポート指定

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { Hono } from 'hono'

  const app = new Hono()
  app.get('/', (c) => c.text('Hello Node.js!'))

  const fetch = app.fetch
  const port = 8080

  serve({
    fetch,
    port,
  })
  ```

- json を返す

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { Hono } from 'hono'

  const app = new Hono()
  app.get('/api/hello', (c) => {
    return c.json({
      ok: true,
      message: 'Hello Hono!',
    })
  })

  serve(app)
  ```

- シャットダウン動作指定

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { Hono } from 'hono'

  const app = new Hono()
  app.get('/', (c) => c.text('Hello Node.js!'))

  const server = serve(app)

  process.on('SIGINT', () => {
    server.close()
    process.exit(0)
  })

  process.on('SIGTERM', () => {
    server.close((err) => {
      if (err) {
        console.log(err)
        process.exit(1)
      } else {
        process.exit(0)
      }
    })
  })
  ```

- 静的ファイル指定

  ```text:フォルダ構成
  ./
   +- server.js
   +- static
       +- hello.txt
       +- image.png
  ```

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { serveStatic } from '@hono/node-server/serve-static'
  import { Hono } from 'hono'

  const app = new Hono()

  // server.js の位置から指定
  // URL で static が指定されているため、./static/* を見る
  app.use('/static/*', serveStatic({ root: './' }))
  app.get('/', (c) => c.text('Hello Node.js!'))

  serve(app)
  ```


### ルータ

https://hono.dev/docs/api/routing

- HTTP メソッド

  ```javascript
  app.get('/',    (c) => c.text('GET /'))
  app.post('/',   (c) => c.text('POST /'))
  app.put('/',    (c) => c.text('PUT /'))
  app.delete('/', (c) => c.text('DELETE /'))

  app.all('/hello', (c) => c.text('Any Method /hello'))

  app.on(['PUT', 'DELETE'], '/post', (c) => {
    return c.text('PUT or DELETE /post')
  })
  ```

- ワイルドカード

  ```javascript
  app.get('/wild/*/card',    (c) => {
    return c.text('GET /wild/*/card')
  })
  ```
