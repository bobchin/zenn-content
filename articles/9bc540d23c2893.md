## まとめ

どんな javascript ランタイム上でも動作するウェブアプリケーションフレームワーク

### インストール

```bash:bash
# インストール
npm create hono@latest my-app
cd my-app
npm install

# npm create hono@latest
# = npm init hono@latest (create は init の alias)
# = npm exec create-hono@latest


# ローカルサーバ実行
npm run dev
```

### まずはじめに

- シンプル

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { Hono } from 'hono'

  const app = new Hono()
  app.get('/', (c) => c.text('Hello Node.js!'))

  serve(app)
  ```

- 起動ポート指定

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { Hono } from 'hono'

  const app = new Hono()
  app.get('/', (c) => c.text('Hello Node.js!'))

  const fetch = app.fetch
  const port = 8080

  serve({
    fetch,
    port,
  })
  ```

- json を返す

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { Hono } from 'hono'

  const app = new Hono()
  app.get('/api/hello', (c) => {
    return c.json({
      ok: true,
      message: 'Hello Hono!',
    })
  })

  serve(app)
  ```

- シャットダウン動作指定

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { Hono } from 'hono'

  const app = new Hono()
  app.get('/', (c) => c.text('Hello Node.js!'))

  const server = serve(app)

  process.on('SIGINT', () => {
    server.close()
    process.exit(0)
  })

  process.on('SIGTERM', () => {
    server.close((err) => {
      if (err) {
        console.log(err)
        process.exit(1)
      } else {
        process.exit(0)
      }
    })
  })
  ```

- 静的ファイル指定

  ```text:フォルダ構成
  ./
   +- server.js
   +- static
       +- hello.txt
       +- image.png
  ```

  ```javascript:server.js
  import { serve } from '@hono/node-server'
  import { serveStatic } from '@hono/node-server/serve-static'
  import { Hono } from 'hono'

  const app = new Hono()

  // server.js の位置から指定
  // URL で static が指定されているため、./static/* を見る
  app.use('/static/*', serveStatic({ root: './' }))
  app.get('/', (c) => c.text('Hello Node.js!'))

  serve(app)
  ```

- リクエストとレスポンス

  - [HonoRequest](https://hono.dev/docs/api/request) を参照

  - c.req.param(): パスパラメータ
  - c.req.query(): クエリストリング
  - c.req.queries(): 複数の値を持つクエリストリング
  - c.req.json(): リクエスト内容をjsonで取得
  - c.req.text(): リクエスト内容をtextで取得
  - c.req.valid('json'): バリデートされたデータを返す

    ```js
    app.get('/posts/:id', (c) => {
      const page = c.req.query('page')
      const id = c.req.param('id')
      return c.text(`You want to see ${page} of ${id}`)
    })
    ```

  - c.status(404): HTTPステータスコードをセットする
  - c.header('Content-Type', 'application/json'): ヘッダーをセットする
  - c.text(): テキスト(`Content-Type: text/plain`)を返す
  - c.json(): json(`Content-Type: application/json`)を返す
  - c.html(): HTML(`Content-Type: text/html`)を返す

  - c.notFound(): not found を返す
  - c.redirect(): リダイレクトする

    ```js
    app.get('/', (c) => {
      return c.html('<h1>Hello! Hono!</h1>')
    })
    app.get('/say', (c) => {
      return c.text('Hello!')
    })
    app.get('/api', (c) => {
      return c.json({ message: 'Hello!' })
    })

    app.get('/notfound', (c) => {
      return c.notFound()
    })

    // 302 を返す
    app.get('/redirect', (c) => {
      return c.redirect('/')
    })

    // 301 で返す
    app.get('/redirect-permanently', (c) => {
      return c.redirect('/', 301)
    })
    ```

### ミドルウェア

https://hono-ja.pages.dev/docs/guides/middleware

  - app.use() で指定する
  - 指定した上から順に実行される

  ```js
  import { basicAuth } from 'hono/basic-auth'
  app.use('/admin/*', basicAuth({
      username: 'admin',
      password: 'secret',
    })
  )

  app.get('/admin', (c) => {
    return c.text('You are authorized!')
  })
  ```

- 作成方法
  - 関数を作成する
    - 第１引数: Context
    - 第２引数: nextメソッド

  - 関数に処理を記述
    - next() を呼ぶことで次のミドルウェアが実行される。 **必ず実行する必要がある。**
    - next() の前後の記述で実行タイミングが異なるので注意

  ```js
  app.use(async (c, next) => {
    console.log('start!')
    await next()
    console.log('end!')
  })
  ```


### ルータ

https://hono.dev/docs/api/routing

- HTTP メソッド

  ```javascript
  // 個別指定
  app.get('/',    (c) => c.text('GET /'))
  app.post('/',   (c) => c.text('POST /'))
  app.put('/',    (c) => c.text('PUT /'))
  app.delete('/', (c) => c.text('DELETE /'))

  // 全てのメソッド
  app.all('/hello', (c) => c.text('Any Method /hello'))

  // 複数メソッド指定
  app.on(['PUT', 'DELETE'], '/post', (c) => {
    return c.text('PUT or DELETE /post')
  })
  ```

- ワイルドカード

  ```javascript
  app.get('/wild/*/card', (c) => {
    return c.text('GET /wild/*/card')
  })
  ```

- 複数のパス指定

  ```javascript
  app.on('GET', ['/hello', '/ja/hello', '/en/hello'], (c) => {
    return c.text('Hello')
  })
  ```
