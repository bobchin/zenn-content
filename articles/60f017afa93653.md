## 使い方

- Arduino IDE
  - ボードマネージャに UIAPduino を追加する
    - 「ファイル」 - 「基本設定」 - 「追加のボードマネージャ」に以下を追記
      `https://github.com/YuukiUmeta-UIAP/board_manager_files/raw/main/package_uiap.jp_index.json`
    - 「ツール」 - 「ボード」 - 「ボードマネージャー」で `uiap` を検索
      `UIAPduino` をインストールする
    - 「ツール」 - 「ボード」 - 「UIAPduino」で、`Pro Micro CH32V003` を選択する

  - プログラムの書き込み
    リセットボタンを押しながらUSB接続。PWRランプがついたらすぐ離す。

  - リセットボタンでモードを変更できるようにする

    ```c
    // 以下を追記
    void setup() {
      if (FLASH->STATR & (1<<14)) NVIC_SystemReset();
      SystemReset_StartMode(Start_Mode_BOOT);
      pinMode(PD4, OUTPUT);
    }
    ```

## サンプル

### LED

  ```c
  #define LED_BUILTIN 2

  void setup() {
    pinMode(LED_BUILTIN, OUTPUT);
  }

  void loop() {
    digitalWrite(LED_BUILTIN, HIGH);
    delay(1000);
    digitalWrite(LED_BUILTIN, LOW);
    delay(1000);
  }
  ```

### スイッチ

- ピン
  - 5V
  - GND
  - GP10(IN)
  - GP02(OUT)

  ```c
  #define LED_PIN 2
  #define SW_PIN 10

  void setup() {
    pinMode(LED_PIN, OUTPUT);
    pinMode(SW_PIN, INPUT);
  }

  void loop() {
    if (digitalRead(SW_PIN) == LOW)
      digitalWrite(LED_PIN, HIGH);
    else
      digitalWrite(LED_PIN, LOW);
  }
  ```

### PWM

  ```c
  const int LED = 2;
  int i = 0;

  void setup() {
  }

  void loop() {
    for (i = 0; i < 4096; i++) {
      analogWrite(LED, i);
      delay(1);
    }
    for (i = 0; i < 4096; i++) {
      analogWrite(LED, 4095 - i);
      delay(1);
    }
  }
  ```
